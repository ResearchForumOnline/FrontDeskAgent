# Industry Playbooks

The app ships as a general front desk console. Add knowledge items for each industry so the agent asks better questions.

## Trades

Capture:

- Name
- Phone
- Postcode
- Job type
- Urgency
- Access details
- Photos or video request
- Preferred appointment window

Escalate:

- Active leaks
- No heating
- Electrical danger
- Lockouts
- Same-day emergency jobs

## Clinics

Capture:

- Patient name
- Contact number
- Appointment type
- Preferred clinician
- Preferred time
- Consent-aware notes

Escalate urgent, clinical, or sensitive issues to a human.

## Education

Capture:

- Student name
- Course or department
- Deadline
- Open day interest
- Admissions question
- Contact details

Route seasonal spikes and international enquiries cleanly.

## Hotels

Capture:

- Guest name
- Stay dates
- Booking reference
- Room or facility question
- Special requests
- Callback details

Escalate payment, complaint, accessibility, and safety issues.
